# Hapi Example

This example starts a [Hapi](https://hapi.dev/) server.

[![Deploy on Railway](https://railway.app/button.svg)](https://railway.app/new?template=https%3A%2F%2Fgithub.com%2Frailwayapp%2Fexamples%2Ftree%2Fmaster%2Fexamples%2Fhapi)

## ✨ Features

- Hapi
- TypeScript

## 💁‍♀️ How to use

- Install dependencies `yarn`
- Connect to your Railway project `railway link`
- Start the development server `railway run yarn dev`

## 📝 Notes

The server started simply returns `"Hello World"`. The server code is located in `src/index.ts`.
